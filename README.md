## Installation

- api server(include mongodb)

```shell
cd api
pip install -r requirements.txt
```

- mongodb for mac

```shell
# Start server
mongod --dbpath ~/data/db
```

- react project

```shell
npm install
npm start
```

## Run Flask Server

```shell
flask run
```
